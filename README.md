# indexcn-hth-lab

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖各类技术演示、工具页面、信息整理与静态功能页面。所有页面均为独立构建，不依赖特定域名或外部服务，便于本地浏览与分发。

仓库名中的 `indexcn` 表示页面主体以中文呈现，`hth-lab` 代表本仓库作为实验性归档空间使用。

## 目录说明

```
/
├── README.md          # 本文件
├── pages/             # 存放所有独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/            # 页面共用资源（CSS、JS、图片等）
│   ├── styles/
│   ├── scripts/
│   └── images/
└── docs/              # 相关说明文档或笔记（可选）
```

- `pages/`：每个 HTML 文件均为一个独立页面，可直接在浏览器中打开。
- `assets/`：存放页面共用的样式、脚本与图片资源，便于统一维护。
- `docs/`：可放置页面说明、更新日志或使用笔记。

## 页面归档说明

- 所有页面均为静态 HTML，无后端依赖，支持离线查看。
- 页面内容以中文为主，部分可能包含少量英文或代码示例。
- 归档遵循“一页一功能”原则，便于索引与引用。
- 页面可能包含实验性功能或技术演示，不保证长期兼容所有浏览器。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `pages/` 目录下的 HTML 文件。
3. 可通过 GitHub Pages 或其他静态托管服务部署 `pages/` 目录。
